# getting started with solo-swap

## prerequisites

- a solana-compatible wallet (phantom, solflare, etc.)
- sol for transaction fees
- tokens you wish to swap

## step 1: connect your wallet

1. navigate to the [solo-swap](https://solo-swap-v2.vercel.app) interface
2. click the "connect wallet" button
3. select your wallet provider from the list
4. authorize the connection request

## step 2: view your tokens

once connected, solo-swap will automatically:
- scan your wallet for all spl tokens
- fetch current market prices
- calculate token values
- display your portfolio in an organized table

## step 3: initial configuration

configure your basic preferences:
- default slippage tolerance
- priority fee settings
- display preferences
