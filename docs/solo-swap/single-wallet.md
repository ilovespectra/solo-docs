# single wallet swapping

## viewing and organizing tokens

### token display options
- **list view**: see all tokens with basic information

### sorting options
click any column header to sort by:
- **token value (usd)**: total value of holdings
- **quantity**: number of tokens held
- **price**: current market price

### filtering
use the search bar and filters to:
- search by token name or symbol
- filter by token type
- show/hide small balances
- isolate specific token categories

## configuring swaps

### 1. select output token
- choose from popular tokens (sol, usdc, usdt)
- search for any spl token
- select multiple output tokens for distribution

### 2. set liquidation percentage
- **full liquidation**: 100% of selected tokens
- **partial liquidation**: set custom percentage (1-99%)
- **value-based**: liquidate or swap specific usd value

### 3. price settings
- **market Price**: execute at current market rate

### 4. initiate swap
1. review your swap parameters
2. check estimated output and fees
3. click "Initiate pro-rata swap"
4. confirm transaction in your wallet
5. monitor swap progress in real-time

## swap confirmation
before executing, you'll see:
- estimated output amount
- network fees
- slippage impact
- transaction summary
