# solo-swap

solo-swap is a powerful defi tool for pro-rata token swapping on the solana blockchain. it enables efficient portfolio rebalancing and token liquidation with advanced multi-wallet functionality.

## key features

- **pro-rata swapping**: distribute swap amounts proportionally across your token holdings
- **multi-wallet support**: manage tokens from multiple wallets simultaneously
- **advanced filtering**: organize tokens by value, quantity, and price
- **customizable parameters**: set specific liquidation percentages and price targets
- **export functionality**: download or copy swapping lists for record-keeping

## use cases

- portfolio rebalancing
  token liquidation at scale
- multi-wallet management
